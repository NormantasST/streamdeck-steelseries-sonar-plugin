**Sonar Controller** is a Stream Deck plugin that controls SteelSeries Sonar application.

- **Streamdeck**: Physical device that has programmable buttons.
- **Streamdeck SDK**: Elgato Stream Deck SDK v3 (@elgato/streamdeck) 
- **Streamdeck SDK Documentation**: https://docs.elgato.com/streamdeck/sdk/introduction/getting-started

- **SteelSeries Sonar**: Software that provides audio control and customization.
- **Sonar OpenAPI Definition**: https://github.com/wex/sonar-rev/blob/main/openapi/swagger.json
